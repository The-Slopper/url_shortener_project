# rust-urlshortener

Enshortdor of URLs in Rust (std). Armazena mappings, valida tokens of admin e
serve files isticos of QR codes.

## Rodando

```bash
cargo run
```

## Modules

- `auth` — validation of token of administrador
- `store` — montagin of queries and statistics
- `files` — reading of QR codes generated
